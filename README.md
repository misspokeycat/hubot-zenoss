# hubot-zenoss

Hubot talks to Zenoss

See [`src/zenoss.coffee`](src/zenoss.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-zenoss --save`

Then add **hubot-zenoss** to your `external-scripts.json`:

```json
[
  "hubot-zenoss"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
